# AI 广告推荐信息流 Android App (Tictok Advert) PRD

## 1. 产品概述

Tictok Advert 是一个面向演示和验收场景的 Android 广告信息流应用。产品重点不是广告投放链路本身，而是展示一个完整的“广告浏览 + 语义搜索 + 统计看板 + 离线素材包”闭环。

## 2. 产品目标

- 提供可滚动、可切频道的广告信息流体验
- 支持大图、小图、视频三种广告卡片
- 在无网络或 AI 服务异常时仍然可完成搜索演示
- 使用本地打包素材保证演示稳定性

## 3. 功能范围

### P0

- 三频道信息流：`精选`、`电商`、`本地`
- 三种卡片样式：大图、小图、视频
- 广告详情页
- Room 本地持久化
- 埋点统计页
- 离线广告包导入

### P1

- Qwen3-32B 语义搜索
- 本地关键词 fallback 搜索
- 标签过滤
- 视频进度续播

### P2

- 脚本化更新广告素材
- 自动替换新的离线广告包

## 4. 当前交付内容

当前仓库已经交付：

- 一个可编译运行的 Android 客户端
- 一个内置的 100 条中文广告离线数据包
- 一个今日头条搜索抓取脚本 `scripts/fetch_toutiao_ads.py`
- 一个 Stats 看板，用于验证曝光、点击和交互数据

## 5. 关键交互

### 首页

- 用户可在频道间切换
- 可通过标签缩小当前频道下的广告范围
- 视频卡片在 feed 中单实例播放

### 搜索页

- 输入自然语言后，优先走 Qwen3-32B 解析
- Qwen 不可用时，自动切换到本地关键词规则
- 搜索结果可直接回看广告详情

### 详情页

- 展示广告封面/视频、摘要、标签和行为按钮
- 从 feed 进入详情页时可承接视频播放进度

### 统计页

- 展示曝光、点击、交互统计
- 支持清空统计数据，便于重复验收

## 6. 交付说明

- 当前仓库交付包含完整代码、README、技术设计文档、学习总结和加分材料
- 演示视频不纳入当前仓库

## 7. 非目标

- 不实现真实广告投放、竞价或服务端回传
- 不保证所有视频关键词都能稳定抓到可直接下载的视频源
- 不把今日头条搜索结果当作正式商业数据源，仅用于演示素材更新
